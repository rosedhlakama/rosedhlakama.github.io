Explain control flow and loops using an example process from everyday life.


Control flow is the order in which the computer executes statements in a script. By default, code is executed from the first line to the last line. That is unless the computer encounters conditionals, breaks or loops. Loops are a way of repeating an action again and again as long as the specified condition remains true. In the movie Groundhog Day, weatherman Phil Connors repeat the same day over and over again. He even dies on some of those day but wakes-up in the morning to relive the day again. He is stuck in a time loop. Its like the universe was running the code for Groundhog Day on a loop until a condition was met. Phil only escaped the loop when he met that condition. It’s a love story of course. When he fell in love and became a better person the loop was broken.

Describe what the DOM is and an example of how you might interact with it


DOM is an abbreviation for Document Object Model, and it is the means by which JavaScript interacts with objects within an HTML document. It’s a programming interface. It is the most important component for working with HTML documents. Its is a hierarchical structure that browsers uses to make sense of and produce the awesome web pages you use daily. Like JavaScript the DOM is object oriented and can be modified with a scripting language. HTML, CSS and JavaScript are equal partners in the creation of the web and what you see in your browser. HTML is the contents of your page and its structure. Simply what you can see is all mostly HTML. CSS is the styling language that gives HTML some style and appeal. CSS is the Karl Lagerfeld of the web developer world. The DOM is the super glue that brings all languages together to create a functional webpage. The DOM is a powerful web development tool which I use to create and add JavaScript to HTML elements.

Explain the difference between accessing data from arrays and objects.


An array stores multiple datum in one variable. For example, an array of the first five chapters from the Handmaid’s tails;
Var Handmaid = [‘Night’, ‘Shopping’, ‘Night’, ‘Waiting Room’, ‘Nap’]
Order matters in an array just like the chapters of a book are read in sequence. To access data stored in an array we use indexes starting at 0. So that would mean Handmaid[3] would point to chapter IV ‘Waiting Room’ because [0] refers to ‘Night’. Arrays are mutable and the data contained in an array can be changed with respect to the order of storage using the following commands;
  • .push() – add a new value to the end
  • .pop() – remove last value
  • .shift() – remove the first value
  • .unshift() – add new value to the start
Objects store data in a structured way using the key: value format. They are flexible and allow for arbitrary combination of strings, numbers, Booleans, arrays, functions and objects. For example, an object containing data about my dog;
var myDog = {
"name": "Zitha", // this is a string
"legs": 4,
"tails": 1,
"friends": ["none"], // this is an array
}
Objects are similar to arrays, except that instead of using indexes to access and modify their data, you access the data in objects through what are called properties. There are two ways to access the properties of an object: dot notation (.) and bracket notation ([]), similar to an array. (.) is used when you already know the name of the property you are trying to access. [] is used if the property you are trying to access has a space in its name. bracket notation can still be used on object properties without spaces. myDog.name = ‘Xena’// will access the object and change my dog’s name to Xena.

Explain what functions are and why they are useful.


A function is a code wrapper that groups statements together making your code reusable. A function is first declared then called or invoked. All of the code between the curly braces will be executed every time the function is called.
A function declared
function sayHello(){ document.getElementById("divide").innerHTML = "HELLO!"; }
A function called
function sayHello(){ document.getElementById("divide").innerHTML = "HELLO!"; } sayHello();
Web page output HELLO!
And to disect the code;
  1. Function instructs javascript to treat this block of code between these {} as a function.
  2. sayHello is the actual name of the function.
  3. () is the opening and closing parentheses which would contain the argument.
  4. {} enclose any statements adjacent to that function inside of which is the statement that makes your function actually do something. In this example the content is the instruction to display Hello! in the space with id “divide”.
  5. Lastly is the function call sayHello() which can be viewed as pressing the = button on a calculator
