Margin, Border, and Padding

The difference between Margin, Border, and Padding.

,Borderand Padding are CSS style rules used for layout and positioning of HTML elements on a webpage.

The image to the left, the box model, illustrates the margin, border and padding style rules as layered boxes.
The element to which the changes can be applied is shown in light blue. This section is what would contain the HTML element, text or image. This is also known as the box model and its size, width and height can be adjusted.

Margindescribes the outermost portion of the box model (the yellow space), creating space around an element, outside of any defined borders. Margins offer a simpler way to position elements.
Margins are set using lengths (pixels or em), percentages, or the keyword auto and can have negative values. (see example of the moving heading to the right) There are four margin values;

  1. Margin-top
  2. Margin-right
  3. Margin-bottom
  4. Margin-left
For more on margin check out the CSS tricks

*when using the shorthand as in the yellow box, values must be in this order: top, right, bottom, and left.

The border property is a simple way of drawing lines around elements which is useful for separating different parts of a webpage. Borders can make webpages more readable and add a graphical flair to images.

Border accepts multiple values. You can see a full list on w3schools site. The main values are:

  1. Border-width |Thickness of the border in measured in px, em, rem, vh and vw units.
  2. Border-style |Type of line drawn around the element. Various values include solid, dashed, and dotted. There are more options available.
  3. Color |The color of the border.

The demo to the right illustrates some of the border values. For more on border style properties visit the CSS tricks page.

Padding describes the innermost part of the box model which creates gaps between the element and the box section. It is prettier to leave a gap between the text or element and the edge of the box section. This way text is easier to read.
There are four padding values which can be set by lengths (px or em) percentages. Similar to margins, padding values are;

  1. Padding-top
  2. Padding-right
  3. Padding Bottom
  4. Padding-left
You could either define;
  • all four values,
  • two or three of the values and the others are assumed, or
  • just one value which is then applied to all four padding properties

The Padding properties can impact the width of the element such that any padding added will in turn add to the width of the element. For more on padding check out this helpful information on CSS Tricks.
